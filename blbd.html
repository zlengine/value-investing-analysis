二、业务分析

Blue Bird是美国最大的校车制造商，专门设计和制造运送学生的黄色校车。简单来说，就是美国街头那些标志性的黄色大校车，几乎一半都是Blue Bird生产的。除了传统柴油校车，公司还是电动校车和替代燃料（丙烷、CNG）校车的市场领导者。

2.1 业务板块

业务板块 FY2025营收 占比 说明
校车（Bus） 约$12.3亿 约83% 设计、制造和销售Type C和Type D校车，包括电动、丙烷、CNG和柴油动力
零部件（Parts） 约$2.5亿 约17% 售后零部件销售，通过经销商网络和大型车队客户销售替换零件
合计 $14.8亿 100% FY2025全年营收（截至2025年9月）

2.2 产品组合

  • Type C校车（常规型）：最畅销的校车类型，车头突出，载客36-72人
  • Type D校车（平头型）：前置发动机，载客54-90人，适合大型学区
  • 电动校车（Electric Vision）：零排放电动校车，受益于EPA清洁校车计划补贴
  • 丙烷/CNG校车：替代燃料校车，运营成本低于柴油
  • 特殊需求校车：配备轮椅升降等无障碍设施

2.3 商业模式

  • 政府客户为主：主要客户是公立学区和私营校车运营商，受政府预算驱动
  • 经销商网络：通过约50家独立经销商销售，覆盖全美和加拿大
  • 替代燃料领先：88%校车仍为内燃机，但电动校车订单快速增长（1,100+辆在手订单）
  • 售后零部件高毛利：零部件业务毛利率高于整车，提供稳定现金流

2.4 地理分布

主要市场在美国（约90%+），加拿大占小部分，少量出口至拉丁美洲和中东。

三、业务板块产品竞争力分析

3.1 校车制造产品竞争力分析

1. 可替代性：低

  • 主要竞争者：IC Bus（Navistar/Traton）、Thomas Built Buses（Daimler）、以及小型电动校车制造商Lion Electric
  • 替代体验差距：Blue Bird拥有近百年校车制造历史，在替代燃料校车领域市场份额领先。其电动校车技术成熟，已交付1,100+辆
  • 替代趋势：稳定。校车行业壁垒极高（安全法规、政府采购流程、品牌信任度），新进入者难以撼动

2. 定价权：中等

  • 校车是政府招标采购，价格竞争存在但品牌和可靠性有溢价
  • 电动校车ASP更高（约$35-40万 vs 柴油校车$10-15万），但依赖政府补贴
  • 零部件业务有较强定价权（售后市场锁定效应）

3. 客户粘性：高

  • 学区一旦选定校车品牌，更换成本极高（司机培训、维修配件、安全认证）
  • 政府采购通常倾向于长期合作的供应商
  • 零部件锁定：Blue Bird校车的替换零件主要由Blue Bird供应

3.2 零部件业务产品竞争力分析

1. 可替代性：低

  • Blue Bird校车的专用零部件具有排他性，第三方难以完全替代
  • 零部件毛利率约30%+，远高于整车业务
  • 随着路上Blue Bird校车保有量增长，零部件需求持续增长

四、历史财务指标

4.1 净资产收益率（ROE）

ROE从2022年的负值迅速恢复至2025年的61.6%，反映了公司盈利能力的强劲改善。但需注意ROE极高部分原因是股东权益基数较小（仅$2.98亿）。

4.2 资产负债率（Debt-to-Asset）

资产负债率从2022年的高位持续下降至约15%，公司几乎没有净债务（净债务为负$1.88亿，即现金超过债务）。财务状况极为健康。

4.3 市盈率（PE）

当前PE约17倍，处于历史中等水平。考虑到公司强劲的盈利增长和零净债务，估值相对合理。

5.3 竞争优势可持续性

  • 市场地位稳固：Blue Bird在替代燃料校车领域领先，电动校车市场份额约50%
  • EPA补贴驱动：联邦清洁校车计划提供大量补贴（每辆电动校车最高$375,000），Blue Bird是最大受益者
  • Lion Electric困境：主要电动校车竞争对手Lion Electric财务困难，减少了竞争压力
  • 风险：政府补贴政策可能变化；原材料（钢、铝）价格波动；电动校车技术路线不确定性

6.2 利润分配特点

  • 不派息、不回购：Blue Bird目前不派发股息，也未进行大规模股票回购，利润主要用于偿还债务和积累现金
  • 净现金状况：公司现金$2.76亿，总债务$0.88亿，净现金$1.88亿，财务极为安全
  • 低资本开支：CapEx仅占营收1-1.5%，轻资产运营
  • 自由现金流改善：从FY2022的负值迅速改善至FY2025的$1.53亿，FCF/营收达10.3%
  • 资本配置评价：保守但合理。公司优先修复资产负债表，未来可能启动回购或派息

七、投资小结

7.1 核心优势

  • 校车行业绝对龙头：约40-45%市场份额，品牌认知度极高
  • 电动校车领先：1,100+辆电动校车在手订单，受益于EPA清洁校车计划
  • 财务健康：净现金$1.88亿，零净债务，Altman Z-score 5.42（AAA级）
  • 高ROE：61.6%的ROE反映极高的资本效率
  • 盈利改善趋势：毛利率从4.4%→20.5%，净利率从-1.1%→8.6%，改善显著

7.2 主要风险

  • 政府补贴依赖：电动校车销售高度依赖EPA清洁校车计划补贴，政策变化风险大
  • 周期性：校车需求受市政预算和利率影响，经济衰退时学区可能推迟更换
  • 原材料成本：钢、铝等原材料价格波动影响毛利率
  • 规模有限：校车市场规模有限（年约4万辆），增长天花板明显
  • 做空压力：约11%的流通股被做空，市场存在分歧

7.4 芒格式评估

业务可理解性：高。制造校车卖给学区，业务模式简单明了。

竞争优势持久性：中高。安全法规壁垒+品牌信任+转换成本构成坚实护城河，但市场规模有限。

管理层资本配置：尚可。目前保守策略合理（修复资产负债表），但长期需要证明资本配置能力。

安全边际：当前PE约17x，P/FCF约15x，对于一家零净债务、高ROE的细分龙头来说估值合理，但安全边际不够充分。理想买入价位在PE 13x以下（约$50-55）。
