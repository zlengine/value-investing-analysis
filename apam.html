二、业务分析

公司简介——用最简单的话说

Artisan Partners是一家帮别人管钱的公司。你把钱交给它，它的专业投资团队帮你买股票、买债券，赚了钱按比例收管理费。就像你请了一个专业园丁帮你打理花园，园丁不拥有花园，但按花园大小每月收服务费。花园越大，收费越多；花园缩小，收费也减少。Artisan的特殊之处在于：它不是一个统一的团队，而是由多个独立的"精品投资团队"组成，每个团队像独立工作室一样运作，有自己的投资风格和决策权。

商业模式

Artisan的收入几乎完全来自管理费——按AUM（管理资产规模）的百分比收取。加权平均费率约69个基点（0.69%），远高于行业多元化资管公司的平均水平。小部分收入来自业绩报酬（FY2025为$2,910万，仅占营收2.4%）。这意味着：

  • 收入与AUM规模直接挂钩——市场涨、客户资金流入，收入增加；反之减少
  • 成本结构高度可变——约48.6%的收入用于员工激励薪酬，随收入浮动
  • 资本需求极低——无需工厂、库存或大量资本开支，几乎100%利润转化为自由现金流

三、业务板块产品竞争力分析

1. 价值/国际价值策略 产品竞争力分析

1. 可替代性：中等

  • 主要替代品：被动指数基金（Vanguard International Value ETF、iShares MSCI EAFE ETF）、其他主动价值基金（TROW、Dodge & Cox）、量化策略
  • 替代品体验差距：Artisan International Value策略10年年化超额收益243bps，长期显著跑赢基准，被动产品无法复制这种alpha
  • 替代品价格差距：被动基金费率5-15bps vs Artisan约65-70bps，价差50-60bps
  • 替代趋势：加速——主动管理行业整体持续被被动替代，APAM连续8个季度净流出

2. 客户转换成本：中等

  • 财务转换成本：机构客户可能面临税务影响（资本利得实现），但共同基金赎回通常无罚金
  • 时间转换成本：机构客户更换管理人需经过投资委员会审批、咨询顾问评估，流程2-3个月
  • 捆绑转换成本：单一策略独立运作，无多策略捆绑锁定
  • 综合评估：机构客户有一定摩擦，但中介渠道（占60%）几乎零摩擦

3. 定价权：中等

  • 历史提价记录：加权平均费率从2018年约62bps逐步提升至2025年69bps，但主要来自策略组合变化而非直接提价
  • 提价vs通胀：费率提升跑赢CPI，但面临行业下行压力
  • 定价权来源：长期超额收益记录（99%策略自成立以来跑赢基准）
  • 定价权趋势：削弱——被动投资持续挤压主动管理费率
  • 巴菲特测试：若管理费提价10%（从69bps到76bps），大量价格敏感的中介渠道客户会加速流出

综合竞争力评级：⭐⭐⭐

2. 成长/全球股票策略 产品竞争力分析

1. 可替代性：中等

  • 主要替代品：成长型ETF（QQQ、VGT）、其他主动成长基金（TROW、Janus Henderson）
  • 替代品体验差距：Global Opportunities等旗舰策略长期跑赢，但近期1年表现落后
  • 替代趋势：加速——成长股ETF尤其受被动化冲击

2. 客户转换成本：中等

  • 与价值策略类似，机构客户有一定摩擦，中介渠道几乎零摩擦
  • 旗舰策略Global Opportunities已关闭新投资者——容量管理保护alpha但限制增长

3. 定价权：中等

  • 旗舰策略关闭新资金是定价权的一种体现——供不应求
  • 但成长策略整体面临更大被动替代压力
  • 定价权趋势：稳定——容量关闭策略维持了费率纪律
  • 巴菲特测试：提价10%可能加速部分客户转向被动替代品

综合竞争力评级：⭐⭐⭐

3. 信贷策略 产品竞争力分析

1. 可替代性：低

  • 主要替代品：高收益债ETF（HYG、JNK）、其他信贷基金、直接贷款基金
  • 替代品体验差距：Artisan信贷团队100%策略在所有期间跑赢基准，主动信贷alpha比股票alpha更难被动复制
  • 替代趋势：稳定——信贷市场效率较低，主动管理仍有明显优势

2. 客户转换成本：中等

  • 信贷策略流动性较低，转换可能面临赎回限制
  • 机构客户对信贷管理人选择更谨慎，评估周期更长

3. 定价权：强

  • 信贷策略费率较高（High Income策略约80-90bps），且客户对费率敏感度低于股票策略
  • 100%策略跑赢基准的记录赋予强定价权
  • 定价权趋势：增强——连续13个季度净流入证明客户认可
  • 巴菲特测试：提价10%不太可能导致大量客户流失，因为alpha远超费率

综合竞争力评级：⭐⭐⭐⭐

4. 另类/新兴市场策略 产品竞争力分析

1. 可替代性：低

  • 主要替代品：新兴市场ETF（EEM、VWO）、其他EM主动基金
  • 替代品体验差距：EM市场效率低，主动管理超额收益空间大；Sustainable EM策略1年回报43.9%
  • 替代趋势：稳定——EM主动管理仍有生存空间

2. 客户转换成本：中等

  • EM策略波动大，客户切换意愿受短期业绩影响显著
  • Antero Peak等对冲策略有锁定期

3. 定价权：强

  • EM和另类策略费率较高（含业绩报酬），且超额收益显著
  • 定价权趋势：增强——EMsights和Antero Peak等新团队正在获得市场认可
  • 巴菲特测试：EM策略的alpha远超费率，提价10%不会导致客户流失

综合竞争力评级：⭐⭐⭐⭐

关键趋势解读：
1. ROE从异常高值逐步正常化：FY2019的370%降至FY2025的155%，原因是股东权益从负值/极低值逐步增长至正常水平。155%的ROE在资管行业中仍属顶尖水平。
2. 营业利润率高度稳定：调整后营业利润率在32%-45%之间波动，FY2021牛市时达45%峰值，FY2023市场低迷时降至32%。FY2025回升至35.3%。
3. 收入高度周期性：营收从FY2020的$8.78亿→FY2021的$11.87亿（+35%）→FY2023的$9.40亿（-21%）→FY2025的$12亿（+28%），与市场走势高度相关。
4. 负债率极高但无实质风险：负债/总资产~86-98%，但负债主要是TRA（税收应收协议）负债$3.03亿和少量债务$1.9亿，无实质性偿债风险。
5. PE估值持续偏低：历史PE在8-15x之间，当前~9.5x处于历史低位，反映市场对主动管理行业结构性风险的定价。

Artisan处于主动资产管理行业，面临双重竞争压力：一是来自同行的主动管理竞争，二是来自被动投资的生存性威胁。

经济护城河评估

品牌力/投资业绩记录：99%策略自成立以来跑赢基准，10年期92%AUM跑赢基准。这是Artisan最核心的护城河——长期alpha记录在行业中极为稀缺。但这是"软护城河"，过去业绩不代表未来。
精品团队模式：自主投资团队结构吸引了顶尖投资人才，过去10年零投资团队离职。这种文化难以复制，但也导致薪酬成本占比高达48.6%，限制了利润率扩张空间。
机构客户转换成本：养老金和捐赠基金更换管理人需经过投资委员会和咨询顾问评估，流程2-3个月。但这只是"摩擦"而非"壁垒"——APAM连续8个季度净流出证明客户仍在离开。
被动投资替代：这是最大的结构性风险。主动管理行业整体在萎缩，Artisan即使业绩优秀也无法独善其身。FY2025股票策略净流出$156亿，仅部分被信贷和另类流入$29亿抵消。
关键人风险：每个投资团队由少数几位基金经理主导，若核心人物离职（虽然10年未发生），对应策略AUM可能迅速流出。
规模劣势：相比BlackRock、TROW等巨头，Artisan缺乏科技平台（如Aladdin）、ETF分销能力和全球品牌影响力，在争夺机构委托时处于劣势。

护城河综合评级

窄护城河——投资业绩记录和团队文化构成差异化优势，但在被动投资大潮下护城河正在被侵蚀。这不是"无法被颠覆"的护城河，而是"在特定细分市场有竞争力"的窄护城河。

利润分配分析要点

高股息回报策略：Artisan将约50-55%的净利润以股息形式分配给股东，加上特别股息，实际回报率更高。当前TTM股息率约8.8%，在低利率环境下极具吸引力。
资本轻模式下的合理分配：资管公司几乎不需要资本再投入（无CapEx、无库存），因此高比例分红是合理的资本配置。留存利润主要用于种子投资（$1.4亿）、新团队孵化和TRA付款。
股息可持续性风险：FY2022年股息支付率高达85.8%，当年净利润$2.24亿但派息$1.92亿。若市场大幅下跌导致AUM缩水，净利润可能不足以维持当前股息水平。
特别股息的灵活性：Artisan通过特别股息在好年景多分配、差年景少分配，比固定高股息更灵活。FY2022特别股息$2.50/股（牛市尾声），FY2023降至$0.75（市场低迷），体现了管理层的审慎。

可持续增长率

可持续增长率 = ROE × 留存比率 = 155% × 41.4% ≈ 64%

这个数字看似极高，但需要注意：Artisan的ROE被低权益基数放大，实际内生增长取决于AUM增长（市场表现+净流入）。当前净流出意味着实际增长低于可持续增长率。

关键风险

1. 主动管理结构性萎缩：过去10年主动基金净流出$3万亿，被动基金净流入$6万亿。即使Artisan业绩优秀，行业趋势不可逆。连续8个季度净流出是这一风险的直接体现。
2. 业绩反转风险：投资业绩是Artisan的生命线。若核心策略（International Value、Global Opportunities）持续跑输基准2-3年，AUM可能加速流出，形成"业绩差→流出→收入降→人才流失→业绩更差"的恶性循环。
3. 关键人风险：虽然10年零团队离职，但每个策略高度依赖少数基金经理。核心人物离职可能导致对应策略AUM大幅流出。
4. 股息可持续性：8.8%的TTM股息率看似诱人，但在市场大幅下跌时可能被迫削减。FY2022年85.8%的支付率已敲响警钟。

关键催化剂

1. 信贷/另类策略加速增长：信贷团队连续13个季度净流入，若信贷+另类AUM占比从16%提升至25%+，可对冲股票流出并提升整体费率。
2. 新团队孵化成功：Grandview房地产PE、EMsights等新团队若成功募资，可打开新增长曲线。
3. 市场回暖驱动AUM增长：AUM每增长10%，营收增加约$8,300万，净利润增加约$2,800万（35%利润率×税率）。
4. ETF/新载体扩张：公司开始提供ETF、SMA等新载体形式，可能吸引新一代投资者。

"这是一家用人才代替资本的公司，赚的是脑力钱而非机器钱。99%策略跑赢基准的记录令人印象深刻，但主动管理正在被指数基金吞噬——这是不可逆的结构性趋势。Artisan就像一位优秀的马车制造商，马车做得再好，也挡不住汽车时代的到来。8.8%的股息率很有吸引力，但你要问自己：这匹马还能跑多久？如果信贷和另类策略能成功转型，这可能是一只能持续下金蛋的鹅；如果不能，这就是一根还剩几口的雪茄烟蒂。"

投资评级

谨慎买入

理由：PE 9.5x + 股息率8.8%提供了足够的安全边际，即使AUM零增长，当前估值也较为合理。信贷/另类策略的增长是关键变量——若成功转型，估值有50%+上行空间；若股票流出加速且新策略增长不及预期，下行风险约20%。适合能承受周期性波动、追求高股息收入的投资者。
