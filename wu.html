二、业务分析

2.1 公司是做什么的？

用最简单的话说：西联汇款是一家"帮人跨国寄钱"的公司。当在美国打工的墨西哥人想把钱寄回家、当在英国工作的印度人想给父母汇款时，西联汇款就是他们最可能选择的渠道之一。

西联汇款的核心模式是"全球代理网络+多渠道接入"：在全球200多个国家和地区拥有约36万个代理点（便利店、邮局、小商店等），客户可以在线下代理点用现金汇款，也可以通过网站和手机App在线汇款。收款人可以在全球任何一个西联代理点凭身份证明取现。

西联汇款的收费模式是"汇款手续费+汇率加价"：客户每汇一笔钱，西联收取固定手续费（约$5-$50不等，取决于金额和渠道），同时在汇率上加价（通常比市场汇率高3-5%）。这就是西联所谓的"take rate"（费率），大约为3.5%——远高于Wise的0.52%。

2.5 商业模式

西联汇款的商业模式核心是"全球网络效应"：

  • 汇款端：36万+代理点 + 网站/手机App，覆盖200+国家
  • 收款端：70M+银行账户直汇 + 300+种取款方式 + 36万+现金取款点
  • 毛利结构：毛利率约37%（持续下降，2021年为43%），运营利润率约18-20%
  • 资本效率：轻资产模式，资本支出极低，自由现金流转化率高
关键运营指标：
  • 2025年全年收入：$40.5亿（同比-3.8%）
  • 2025年GAAP净利润：$5.0亿（同比-46.5%，2024年有税收优惠）
  • 2025年调整后EPS：$1.75
  • 品牌数字交易量增长：+12%（2025年），+21%（2026Q1）
  • 数字渠道占CMT交易量：42%（2026Q1）
  • 消费者服务收入占比：13%（2025年），从2022年的6%大幅提升
  • 运营现金流：$5.44亿（2025年）
  • 自由现金流：约$5亿（2025年）

ROE分析：西联汇款的ROE数据非常具有误导性。2018-2020年ROE出现极端负值或超高值，是因为公司通过大规模回购将股东权益压到了接近零甚至负数。当股东权益只有几亿美元时，即使净利润只有几亿，ROE也会高达100-400%。

芒格会说："当一家公司的ROE超过50%时，不要急着鼓掌——先看看它的股东权益是不是被回购压到了接近零。高ROE可能是优秀经营的标志，也可能是财务工程的结果。西联汇款属于后者。"

全球跨境汇款市场规模约$2万亿（C2C），预计2032年增长至$3.1万亿。市场参与者分为三类：

4.2 西联汇款的竞争护城河

1. 全球代理网络（Global Agent Network）——最核心的护城河
西联汇款拥有约36万个代理点，覆盖200+国家和地区。这个网络花了170多年建立，是无法被快速复制的物理基础设施。对于没有银行账户的汇款人（全球约14亿人），现金汇款是唯一选择，而西联是覆盖最广的选择。
  • 约90%的代理点在美国以外
  • 4.6亿银行账户可直汇
  • 300+种取款/支付方式
2. 品牌认知度（Brand Recognition）
"Western Union"在汇款领域就像"Kleenex"在纸巾领域——它是品类代名词。对于许多发展中国家的移民工人，西联汇款是他们知道的第一个也是唯一一个汇款品牌。这种品牌心智在低收入、低教育人群中尤其强大。
3. 合规与监管牌照
在200+国家运营需要获得当地的汇款牌照和合规审批。西联拥有全球最完整的汇款合规体系，这是新进入者需要数年才能建立的壁垒。但合规成本也是西联费用高的原因之一。
4. 全渠道能力（Omnichannel）
西联是唯一同时拥有线下代理网络和线上数字渠道的汇款公司。客户可以在线下汇款、线上收款，或反之。这种"线上+线下"的灵活性是纯数字玩家（如Wise、Remitly）无法提供的。

4.3 护城河的可持续性评估

护城河强度：中等偏弱（5/10）——正在被侵蚀

西联汇款的护城河是真实的，但正在被结构性侵蚀：
  • 数字替代：Wise的take rate仅0.52% vs 西联3.5%，价格差距6-7倍。随着银行账户普及率提升（2024年全球79% vs 2014年62%），越来越多的人选择更便宜的数字渠道
  • 现金汇款萎缩：全球移动支付账户已达15%，撒哈拉以南非洲40%成年人拥有移动支付账户。现金汇款是西联的根基，但这个根基正在缩小
  • 走廊专用竞争者：Remitly专注美-菲、美-印等高利润走廊，以更好的用户体验蚕食西联份额
  • 美国移民政策风险：2025年美国移民执法加强导致美-墨走廊汇款量骤降16.2%，这是西联最大的收入来源之一
  • 美国汇款税：One Big Beautiful Bill法案将对现金汇款征收1%的税，可能加速从现金向数字渠道的转移
芒格会说："西联汇款的护城河就像一座古老的城堡——城墙还在，但护城河里的水正在被抽干。数字技术正在改变汇款的方式，而西联的代理网络优势在数字世界中毫无用处。除非它能成功转型为数字优先的公司，否则这座城堡终将被攻破。"

5.2 股东回报亮点

资本配置评估：中等偏下（5/10）

正面：
  • 持续分红：自2006年上市以来持续分红，从未削减。当前年化$0.94/股，股息率约10.4%
  • 积极回购：过去10年累计回购约$40亿+，流通股从5.17亿股降至3.12亿股（减少40%）
  • 2024年12月新批$10亿回购计划（无到期日）
  • 2025年回报$5.3亿，占自由现金流的106%
负面：
  • 过度回购：2017-2020年间回购力度过大，导致股东权益被压至负值。这是典型的"杀鸡取卵"——为了短期EPS增长牺牲了财务安全
  • 分红零增长：自2021年起，季度分红一直维持在$0.235/股不变，已连续5年零增长。考虑到10%+的股息率，不增长可能意味着管理层对未来没有信心
  • 回报超过自由现金流：2024年和2025年股东回报/FCF超过100%，意味着公司在借钱分红和回购
  • 增长投入不足：资本支出极低（每年约$3-4亿），大部分是维护性支出。在数字转型关键期，投入明显不足
巴菲特会说："一家公司如果把所有现金流都用来分红和回购，而不是投资于未来增长，那它就是在告诉市场：'我们看不到好的投资机会。'对于西联汇款来说，这可能是诚实的——但也是危险的。在数字汇款时代，不投资就意味着缓慢死亡。"

六、战略转型：BEYOND计划

6.1 从Evolve到BEYOND

2022年，CEO Devin McGranahan启动了"Evolve 2025"战略，目标是恢复增长。2025年11月，公司发布了升级版"BEYOND"战略，目标是在2028年实现收入$48-53亿（中值$50亿），相比2025年的$40.5亿增长约23%。

转型评估：BEYOND战略的方向是正确的——从传统现金汇款转向数字优先、多元化服务。但执行风险很大：
  • 稳定币USDPT：概念很新，但西联不是加密公司，能否成功运营稳定币存疑。Circle（USDC发行方）用了5年才达到$400亿规模
  • Intermex收购：$5亿收购一个本身在衰退的业务（Intermex连续4个季度收入下滑），价格是否合理？
  • 消费者服务：从6%到13%的进展不错，但旅行换汇和账单支付都是低利润率业务
  • 2028年$50亿收入目标：意味着2025-2028年CAGR约7%，而公司已连续5年收入负增长

七、价值投资评估

7.1 巴菲特/芒格式评估

总体评分：4/10（高风险，需谨慎）

1. 业务是否可理解？——是
帮人跨国寄钱，这个业务非常简单。巴菲特能完全理解。

2. 是否有可持续的竞争优势？——有，但正在被侵蚀
全球代理网络是真实的护城河，但数字技术正在绕过这个网络。就像柯达的胶片分销网络一样——曾经是护城河，后来变成了负担。西联的代理网络在现金汇款时代是护城河，在数字汇款时代可能变成成本负担。

3. 管理层是否诚实能干？——尚可
CEO McGranahan自2019年上任以来推动了数字转型，方向正确但进展缓慢。管理层在资本配置上过于激进（回购导致负权益），分红5年零增长也传递了负面信号。

4. 价格是否有安全边际？——有，但有陷阱
当前估值看似便宜：PE 6x，FCF收益率17%，股息率10%。但低估值陷阱的风险很大：
  • 如果营收继续下滑，EPS也会下滑，"低PE"可能变成"高PE"
  • 10%的股息率在历史上往往是"价值陷阱"的信号——市场在定价分红削减的风险
  • 高杠杆意味着衰退时容错空间小

核心矛盾：西联汇款就像一家"正在融化的冰块"——你可以在冰块融化前喝到很多水（高股息+回购），但冰块最终会消失。问题是：融化速度有多快？

三种情景：
  • 乐观情景（20%概率）：BEYOND战略成功，2028年收入达$50亿，数字业务占比超50%，PE修复至10x → 股价$18+（翻倍）
  • 基准情景（50%概率）：收入缓慢下滑或持平，EPS维持$1.5-1.8，PE维持5-7x → 股价$8-13，靠10%股息获得年化10-12%回报
  • 悲观情景（30%概率）：收入加速下滑，被迫削减分红，PE降至3-4x → 股价$5-7，亏损20-45%
巴菲特式忠告："以便宜的价格买入一家正在衰退的企业，这不是价值投资——这是价值陷阱。西联汇款的高股息很诱人，但你需要问自己：5年后这家公司还会存在吗？10年后呢？如果答案是'可能不会以现在的形式存在'，那么10%的股息率可能只是'冰块融化前最后的美味'。真正的价值投资是买入一家正在成长的好公司，而不是买入一家正在衰落的便宜公司。"
