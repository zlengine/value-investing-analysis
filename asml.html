一句话业务：ASML是全球唯一能制造EUV（极紫外）光刻机的公司，市占率100%。EUV光刻机是制造7nm及以下先进芯片的必需设备——没有ASML的机器，台积电、三星、英特尔就无法生产先进AI芯片和手机处理器。一台EUV光刻机售价约1.8亿欧元，重180吨，含10万+零件，运输需要40个集装箱+3架货机+20辆卡车。这是人类制造过的最精密机器。

二、业务分析

2.1 业务模式——用大白话说

想象你要在一张A4纸上画出整座城市的地图——每条街道、每栋楼、每个红绿灯都要精确到毫米。光刻机就是干这个的：把芯片设计师画好的电路图，用光"印"到硅片上。光刻机就像一台超级精密的"投影仪"——把电路图案投影到硅片上，然后用化学方法把图案刻出来。

ASML的特殊之处在于：它用的是"极紫外光"（EUV），波长只有13.5纳米，比传统光刻用的深紫外光（DUV，193纳米）短了14倍。波长越短，能画出的线条越细，芯片上的晶体管就越多、越快、越省电。7nm、5nm、3nm、2nm——这些先进制程芯片，必须用ASML的EUV光刻机才能制造。全球没有任何其他公司能做。

赚钱方式有三块：卖新设备（一次性大单，占75%）、装机量管理（服务+升级+备件，占25%，持续增长）、软件（计算光刻解决方案，嵌入设备中）。

三、产品竞争力分析

3.1 EUV光刻系统 产品竞争力

1. 可替代性：极低

  • 直接替代品：无。全球100%市占率，零竞争对手
  • 间接替代品：DUV多重曝光（可部分替代但良率低、成本高、周期长）；纳米压印NIL（佳能，仅限3D NAND，逻辑芯片不可用）
  • 替代品体验差距：巨大。DUV多重曝光需要4-6次曝光才能实现EUV单次曝光的效果，良率下降20-30%，成本增加50%+
  • 替代趋势：有利于ASML。3nm以下制程DUV多重曝光已物理上不可行，EUV是唯一选择

2. 客户转换成本：极高

  • 财务转换成本：不存在"转换"——因为没有替代品。如果ASML不卖EUV给你，你就无法生产先进芯片
  • 时间转换成本：尼康和佳能已退出EUV研发10年+，即使重新投入也需要10-15年才可能量产
  • 技术转换成本：EUV光刻工艺与芯片设计深度耦合，更换光刻技术意味着重新设计整个芯片

3. 定价权：极强

  • 历史提价记录：EUV系统售价从2018年的约€1.2亿升至2025年的约€1.8亿，High-NA EUV售价€3.5亿+。年化提价5-8%
  • 巴菲特测试：如果ASML涨价10%（EUV从€1.8亿涨到€2亿），客户会离开吗？绝对不会——因为根本没有替代品。台积电、三星、英特尔别无选择
  • 定价权来源：100%垄断 + 先进制程刚需 + 客户零替代选择
  • 定价权趋势：持续增强。High-NA EUV（€3.5亿+）比Low-NA EUV（€1.8亿）贵近2倍，客户仍排队购买

整体竞争力评级：★★★★★ 5/5——这是全球罕见的"真正垄断"，不是"接近垄断"，而是100%市占率、零竞争对手。

3.2 DUV光刻系统 产品竞争力

1. 可替代性：中等

  • 直接替代品：尼康ArF浸没式DUV、佳能i-line/KrF
  • 替代品体验差距：ASML DUV浸没式市占率80%+，尼康产品性能差距明显
  • 替代趋势：稳定。ASML在高端DUV领域优势稳固

2. 客户转换成本：高

  • 财务转换成本：DUV设备€5,000万-€1亿/台，更换供应商需重新验证工艺
  • 时间转换成本：新设备导入验证6-12个月

3. 定价权：强

  • 巴菲特测试：涨价10%客户不会离开，因为ASML DUV性能和稳定性远超尼康
  • 定价权来源：技术领先 + 80%+市占率 + 服务生态

整体竞争力评级：★★★★ 4/5

3.3 装机量管理（服务+升级） 产品竞争力

1. 可替代性：极低

  • 直接替代品：第三方维修商。但ASML对自己的设备最了解，拥有独家诊断软件和AI预测性维护
  • 替代品体验差距：巨大。EUV设备停机1小时损失可达数百万美元，客户不敢用非原厂维护

2. 客户转换成本：极高

  • ASML的NXE:3800E性能升级可将EUV产能提升37%，这种升级只有原厂能做
  • 服务合同与设备绑定，更换服务商意味着放弃升级能力

3. 定价权：极强

  • 服务毛利率极高（估计55%+），远高于设备销售
  • 装机量管理营收FY2025增长26%至€82亿，是增长最快的板块

整体竞争力评级：★★★★★ 5/5

ROE趋势：从FY2016的18.2%持续攀升至FY2025的49.0%，10年增长170%。这主要来自真实经营能力提升（净利率从21.5%升至29.4%）和回购推高权益乘数（1.73→2.20）。即使剔除杠杆效应，ROA也从10.5%升至22.2%，经营质量极为出色。

负债率：持续下降从35%降至20.5%，净现金€83亿。财务安全边际极高，信用评级A+优于AMAT和KLAC。

5.2 经济护城河评估

1. 技术垄断（极宽）
EUV光刻100%市占率——这不是"竞争优势"，这是"绝对垄断"。尼康和佳能已退出EUV研发10年+，即使今天重新投入，也需要10-15年和$100亿+研发才能追上。High-NA EUV（0.55 NA）ASML已开始交付，领先优势进一步扩大到15-20年。这是全球科技行业中最宽的护城河之一。
2. 专利壁垒（极宽）
ASML持有EUV核心专利1,000+项，覆盖光源、光学系统、精密工作台等关键子系统。任何试图进入EUV领域的竞争者都必须绕过这些专利——在实践中几乎不可能。
3. 供应链生态（极宽）
EUV系统含10万+零件，来自800+全球供应商。核心供应商高度绑定：蔡司（光学系统，ASML持有24.9%股权）、Cymer/TRUMPF（光源）、VDL（工作台）。蔡司的光学镜面精度要求：如镜面放大到德国国土面积，最大凸起不超过1毫米——这种精度全球只有蔡司能做到。ASML与蔡司的合作超过25年，10万+工时开发High-NA光学系统。新进入者无法复制这个供应链生态。
4. 客户转换成本（极宽）
先进制程客户验证周期2-3年，光刻工艺与芯片设计深度耦合。台积电、三星、英特尔的整个先进制程产线都围绕ASML EUV设计，更换光刻技术意味着重新设计芯片和产线——成本和风险不可承受。
5. 规模优势（宽）
R&D投入€47亿/年（14%营收占比），远超佳能+尼康研发总和。EUV累计研发投入超€80亿，只有ASML有规模支撑这种投入。但需注意：ASML更像"集成商"而非"制造商"，核心部件依赖蔡司/TRUMPF等独家供应商。
6. 潜在威胁：纳米压印（NIL）
佳能的纳米压印技术（NIL）是唯一非光学替代路线，已在3D NAND领域部署。NIL成本仅为EUV的40%，功耗仅为EUV的10%。但目前缺陷密度问题未解决，逻辑芯片不可用。如果未来5-10年NIL在逻辑芯片上突破，可能对ASML的DUV业务构成威胁——但EUV的垄断地位在可预见的未来无法撼动。

5.3 护城河评级

整体护城河评级：极宽（Very Wide）

ASML拥有全球科技行业中最宽的护城河之一——EUV 100%垄断 + 1,000+核心专利 + 不可复制的供应链生态 + 极高客户转换成本。与AMAT的"广度型护城河"和KLAC的"单项深度护城河"不同，ASML的护城河是"绝对垄断型"——不是"几乎垄断"，而是"完全垄断"。这是巴菲特最梦寐以求的商业模式。

6.2 资本配置评价

股东回报：FY2025向股东返还€85亿（分红€26亿+回购€59亿），占FCF的77%。分红连续增长（FY2025 €7.50/股，同比+17%）。2026年1月宣布新回购计划€120亿（至2028年12月），彰显长期信心。
回购质量：累计回购从2006年的€6.77亿增至2025年的€292.5亿。FY2021在PE 42x时回购€86亿，FY2025在PE 50x+时回购€59亿。与AMAT类似，高估值时大量回购是资本配置的瑕疵——巴菲特更希望在PE 20-30x时回购。
R&D投入：FY2025 R&D €47亿（14%营收占比），远超AMAT的$35亿（12%）和KLAC的$16亿（15%）。这是ASML维持EUV垄断地位的根基——持续的高研发投入确保技术领先优势不断扩大。
可持续增长率：ROE 49% × 留存率 73% = 35.8%。理论上ASML可以不靠外部融资实现36%的增长，但实际营收增速约15-20%，说明大量留存利润被用于回购和R&D，而非业务扩张。这是合理的——EUV产能扩张受供应链瓶颈限制，不是想扩就能扩。

7.2 关键风险

1. 中国出口管制风险：中国曾占ASML营收41%（FY2024），EUV已完全禁售，DUV出口管制持续收紧。美国"MATCH Act"可能进一步限制。CFO Dassen预计2026年中国销售占比降至约20%。虽然非中国需求可抵消，但长期看中国国产替代（SMEE 28nm DUV）将侵蚀ASML在华DUV市场。
2. 客户集中度：台积电、三星、英特尔三大客户占EUV需求90%+。台积电一家的资本开支波动就能显著影响ASML业绩。FY2025 Logic系统营收中，单一客户（台积电）占比极高。
3. 供应链瓶颈：ASML是"集成商"而非"制造商"，核心部件依赖蔡司（光学）、TRUMPF（激光）等独家供应商。蔡司光学系统产能扩张20-25%仍可能跟不上ASML需求。如果核心供应商出问题，ASML无法自行替代。
4. 估值泡沫风险：PE 64x意味着市场预期ASML将长期维持20%+增速。在周期性行业中，这种预期极度乐观。一旦AI投资放缓或EUV出货不及预期，股价可能大幅回调。
5. 技术替代风险（长期）：佳能NIL在3D NAND已部署，如果未来5-10年在逻辑芯片上突破，可能侵蚀DUV市场。先进封装（Chiplet）可能减少对最先进光刻的依赖。但这些风险在5-10年内不会威胁EUV垄断。

7.3 关键催化剂

1. High-NA EUV量产：EXE:5200B已交付英特尔（14A/1.4nm节点），台积电和三星2026年纳入量产路线图。High-NA售价€3.5亿+（Low-NA的2倍），将显著提升ASML营收和毛利率。
2. DRAM EUV渗透加速：三大DRAM厂商（三星、SK海力士、美光）全面转向EUV，FY2025 Q4 Memory订单占比56%。DRAM EUV渗透率仅20-30%，远低于Logic的90%+，增长空间巨大。
3. AI超级周期：CEO Fouquet表示"芯片需求超过供给"，Memory客户"今年已售罄"，Logic客户"加速扩产"。2026年营收指引€360-400亿（+10-22%），EUV出货60台（+25%），2027年产能80台。
4. 订单积压€388亿：提供2-3年营收可见性，即使短期需求波动也不影响交付。FY2025 Q4单季新增订单€132亿（EUV €74亿），创历史新高。
